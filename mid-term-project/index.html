Addendum: Process & Challenges

This Doraemon-themed adventure game was designed to practice HTML, CSS, and JavaScript skills. I planned the story flow with multiple paths to ensure at least 8 unique endings. Each story node includes text, image, and choice objects. Challenges included managing dynamic buttons and making sure images update correctly with story progression. Using a central update function simplified debugging and allowed all paths to work smoothly. This project improved my understanding of DOM manipulation, event listeners, and interactive web design.
