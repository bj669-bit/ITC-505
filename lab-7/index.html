Addendum: This Mad Lib is created by Bhavana Jandhyala for ITC-505 Lab 7.

Difficulties Faced: Initial attempts with a server-side Express setup caused deployment issues on GitHub Pages, which does not support Node.js. We had to convert the project to fully client-side JavaScript.

How We Ran Everything: All functionality, including Mad Lib generation and random number generation, is handled in-browser using JavaScript. No server is required. Simply open the page on GitHub Pages to use the lab.

Additional Notes: The Mad Lib is responsive, interactive, and the footer displays the last modification date automatically.
