Project Addendum

Process: The game was created using a 5×5 grid where each tile toggles itself and its orthogonal neighbors. To ensure solvability, the board begins in an all-off state and is scrambled by simulating random legal moves.

Creative Choices: The UI uses Google Fonts (Space Grotesk & Inter) for a fun, modern "font-crazy" look as required. Buttons maintain accessibility, and the layout is responsive.

Challenges: Ensuring solvability was the main challenge, resolved by scrambling using legal moves only. Minor UI timing fixes ensured the final tile update displays before the win alert.

Insights: Simulated-scramble solves solvability elegantly without matrix math. Accessibility and visual clarity improve gameplay and debugging.
